# requirejs_tutorial
a tutorial for merge multi html pages into a single page, by using require.js

1. require.js引入
	require.js, config, shim
2. require.js简单示例
	加载html, 加载css, 加载js, 查看文件是如何加载
3. require.js 两个页面切换示例
	查看资源是否重复加载
4. 双页面合并过程中的注意点
	1. html 应该包含于一个唯一的容器。
	2. css 作用域应该以容器开头。
	3. js 封装格式。 所有的dom操作的应该包含于容器作用域。
